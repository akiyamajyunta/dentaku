# 概要

電卓です。フロントエンドの勉強がてら作りました。

# 使い方

```
git clone https://github.com/akiyamajyunta/dentaku.git
cd ./dentaku
npm i run 
npm run dev
```

http://localhost:5173/ にアクセス

# 使える機能

1. 四則演算
2. 小数点の計算
3. ()機能
4. 入力した数値を一つづつ消せる
5. =後の計算
6. 円周率π機能

# 【】(かっこ)について
優先して計算を行えます。一回押すと開き、もう一回押すと閉じます。事前に、処理後に行いたい四則のボタンを押してください

市販のやつと何ら変わりません
